
> ## TIP
> 使用`VS Code`开发，样式文件使用`.less`输出`.wxss`文件 \
> 在`VS Code`中安装`Easy LESS`插件，自动编译Less到css保存 \
> 设置(`ctrl + ,`) → 打开设置(`json`) → `settings.json`配置如下：


``` json
{
  // 保存less 输出.wxss文件
  "less.compile": {
    // "compress": true, //是否压缩
    // "sourceMap": false, //是否生成map文件，设置为"true"时可以在调试台看到less行数
    // "main": ["a.less","b.less"],
    // "out": true, //是否输出css文件，false为不输出
    // "out": "${workspaceRoot}\\css\\com",
    // "outExt": ".min.css",
    "outExt": ".wxss", //输出文件的后缀，小程序可以写为"wxss"  ✔
    // out 为true时，输出文件在当前目录, 也可以指定输出目录
    // main 为指定文件可以上 "a.less", 多个用数组的方式 ["a.less", "b.less"]
  },
}
```


> ## TIP
> 微信小程序 设计稿 宽度`750px` \
> 微信小程序中单位使用`rpx`

``` LESS
// 1.设计稿宽度 750px
   750px = 750rpx
   1px = 1rpx
// 2.设计稿宽度 375px
   375px = 750rpx
   1px = 2rpx
   1rpx = 0.5px
// 3.宽度 414 / 未知(page)
   page px = 750rpx
   1px = 750rpx / page
   100px = 750rpx * 100 / page

   width: calc(750rpx * 100 / 375);
```

> ## WARNING
> 开发中微信小程序样式注意 `.less` / `.wxss`

``` LESS
/* 微信小程序中不识别 通配符'*' */
page,view,text,swiper,swiper-itrm,image,navigator{
    padding: 0;
    margin: 0;
    box-sizing: border-box;
}
```

``` LESS
/* 原生css 和 wxss 也是支持变量*/
page{
    /* 定义主题颜色 */
    --themeColor: #fff;
    /* 定义统一字体大小 */
    /* 假设设计稿 大小是375px */
    /* 1px = 2rpx */
    /* 14px = 28rpx */
    font-size: 28rpx;
}

/* 使用 主题颜色*/
view{
    color: var(--themeColer);
}
```
