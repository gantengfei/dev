
> 以下要将一个 `5GB` 的文件 拆分成 每个不超过 `2GB` 的压缩分卷（常用于上传限制、邮件附件、网盘分片等场景）

使用以下方法，具体取决于你的操作系统（Windows / macOS / Linux）和工具偏好。

# ➤ 目标

- ● 原始文件：`largefile.zip`（或其他任意 5GB 文件）
- ● 输出：多个压缩分卷，如 `archive.zip.001`, `archive.zip.002`, ...，每个 ≤ 2GB

> 2GB = 2,147,483,648 字节（约 2 × 1024³）


# 方法一：使用 WinRAR（Windows）

## 1.1 步骤（以 Windows 为例，macOS/Linux 类似）

- ● 右键文件 → “添加到压缩文件...”
- ● 在“分卷大小”中输入：`2048`，单位选 MB（即 2048 MB ≈ 2GB）
- ● 点击确定。

## 1.2 输出示例
``` Text
archive.part1.rar  (2GB)
archive.part2.rar  (2GB)
archive.part3.rar  (1GB)
```

> ## NOTICE
> 注意：**RAR** 的 2048 MB 实际是 2048 × 1024² = 2,147,483,648 字节，正好 2GB。


# 方法二：使用 7-Zip（推荐，跨平台、免费、强大）

## 2.1 步骤（以 Windows 为例，macOS/Linux 类似）

- ● 安装 `7-Zip`（Windows）或通过包管理器安装：
- &emsp;● macOS: `brew install p7zip`
- &emsp;● Ubuntu/Debian: `sudo apt install p7zip-full`
- ● 右键文件 → `7-Zip` → `“Add to archive...”`
- ● 在弹出窗口中设置：
- &emsp;● **Archive format**: `7z` 或 `zip`（建议用 `7z`，支持更好的分卷和压缩）
- &emsp;● **Split to volumes, bytes**: 输入 `2G`（7-Zip 支持单位：`k=KB`, `m=MB`, `g=GB`）
- &emsp;● **Compression level**: 根据需要选择（如“Normal”）
- ● 点击 **OK**，等待完成。

## 2.2 输出示例
``` Text
largefile.7z.001  (2GB)
largefile.7z.002  (2GB)
largefile.7z.003  (1GB)
```

> ## NOTICE
> 如果是敏感文件，可勾选“加密文件名”并设置密码。

## 2.3 命令行方式（通用）
``` Bash
# 使用 7z 分卷压缩（每卷 2GB）
7z a -v2g largefile.7z your_5gb_file.dat
```

- ● `-v2g`：表示 volume size = 2GB
- ● 解压时只需双击 `.001` 文件，或运行：

``` Bash
7z x largefile.7z.001
```


# 方法三：Linux / macOS 命令行（无压缩，仅拆分）
如果你不需要压缩，只是拆分大文件（例如已压缩的视频）：
``` Bash
# 拆分为 2GB 的二进制块
split -b 2G your_5gb_file.dat chunk_

# 输出：chunk_aa, chunk_ab, chunk_ac, ...
```

合并还原：
``` Bash
cat chunk_* > your_5gb_file_restored.dat
```

> ## NOTICE
> 此方法不压缩，仅分割。若需压缩+分卷，请用 `7z`。


# ➤ 重要注意事项

| 项目           | 说明                                                                  |
| :------------- | :-------------------------------------------------------------------- |
| 分卷大小单位   | `2G` = 2 × 1024³ = 2,147,483,648 字节（不是 2×10⁹）                   |
| 文件系统限制   | FAT32 不支持单文件 >4GB，但分卷后每个 <2GB 可安全存储                 |
| 解压要求       | 所有分卷必须在同一目录，且文件名连续（不能重命名）                    |
| 压缩 vs 不压缩 | 如果原文件已是压缩格式（如 .mp4, .zip），再压缩收益小，可直接 `split` |


# ➤ 推荐方案

| 场景                 | 工具                    | 命令/操作                                  |
| :------------------- | :---------------------- | :----------------------------------------- |
| 通用、跨平台、带压缩 | **7-Zip**               | `7z a -v2g output.7z input_file`           |
| Windows 图形界面     | **7-Zip** 或 **WinRAR** | 右键 → 添加到压缩文件 → 设置分卷为 2048 MB |
| 仅拆分（不压缩）     | `split`（Linux/macOS）  | `split -b 2G file chunk_`                  |


# ➤ 安全提示

- ● 分卷文件传输时，建议计算 **MD5/SHA256** 校验和，确保完整性。
- ● 敏感数据请加密（**7-Zip** 支持 **AES-256** 加密）。
