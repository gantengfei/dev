# 删除文件
 `del`或`erase`命令

`del`命令用于删除一个或多个文件

> ## TIP
> `c:\Users\lenovo>del /?` \
> 删除一个或数个文件 \
> `del [/p] [/f] [/s] [/q] [/a[[:]attributes]] names` \
> `erase [/p] [/f] [/s] [/q] [/a[[:]attributes]] names`
>
> `names` 指定一个或多个文件或者目录列表。 \
> 通配符可用来删除多个文件。 \
> 如果指定了一个目录，该目录中的所 \
> 有文件都会被删除。 \
> `/p` 删除每一个文件之前提示确认。 \
> `/f` 强制删除只读文件。 \
> `/s` 删除所有子目录中的指定的文件。 \
> `/q` 安静模式。删除全局通配符时，不要求确认 \
> `/a` 根据属性选择要删除的文件 \
> **属性**  \
> `r` 只读文件     `s` 系统文件     `h` 隐藏文件 \
> `a` 存档文件     `i` 无内容索引文件     `l` 重分析点 \
> `-` 表示“否”的前缀

## 删除一个文件
输入`del 文件名`删除该文件
``` bash
del a.txt
```

## 删除某个后缀的所有文件
输入`del *.后缀名`删除以后该缀名的所有文件，如要删除掉当前目录下所的所有.html文件
``` bash
del *.html
```

删除文件夹中所有子目录中所有.js.map文件
``` bash
del *.js.map /s
```

删除掉当前目录下的所有.html,.js,.css文件
``` bash
del *.html *.js *.css
```
erase命令和del命令的功能一样
``` bash
erase *.html *.js *.css
```
