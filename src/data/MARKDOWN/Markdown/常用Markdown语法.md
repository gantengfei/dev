# 常用 Markdown 基础语法（速查）

本文件为常用 Markdown 语法速查表，包含常用示例与说明，适用于 GitHub 风格（GFM）Markdown。

## 目录
- 标题（Heading）
- 强调（Bold / Italic）
- 列表（无序 / 有序 / 任务列表）
- 链接与图片
- 代码（行内 / 代码块）
- 引用（Blockquote）
- 表格
- 分隔线
- 换行与转义

---

## 标题（Heading）

使用 1 到 6 个 `#` 表示不同层级的标题：

``` md
# 一级标题
## 二级标题
### 三级标题
```

## 强调（Bold / Italic）

- 斜体：`*斜体*` 或 `_斜体_`
- 粗体：`**粗体**` 或 `__粗体__`
- 同时使用：`***加粗并斜体***`

## 列表

- 无序列表使用 `-`、`+` 或 `*`：

``` md
- 第一项
- 第二项
  - 嵌套项
```

- 有序列表使用数字加点：

``` md
1. 第一项
2. 第二项
```

- 任务列表（GFM）：

``` md
- [x] 已完成项
- [ ] 未完成项
```

## 链接与图片

- 行内链接：`[文本](https://example.com)`
- 自动链接：`<https://example.com>`
- 图片：`![替代文本](path/to/image.png "可选标题")`

## 代码

- 行内代码使用反引号：`` `code` ``
- 代码块使用三反引号，并可指定语言以启用语法高亮：

``` js
function hello() {
  console.log('Hello');
}
```

或使用缩进（4 个空格）表示代码块（不推荐用于混合场景）。

## 引用（Blockquote）

使用 `>`：

``` md
> 这是一个引用
> 可以换行
```

## 表格（GFM）

``` md
| 列1 | 列2 | 列3 |
| --- | :--: | --: |
| 左对齐 | 居中 | 右对齐 |
```

对齐：`:` 放在左、右或两侧控制对齐方式。

## 分隔线

使用 `---`、`***` 或 `___`（单独一行）：

``` md
---
```

## 换行与转义

- 段内换行：行尾加两个空格然后回车。
- 转义特殊字符：在特殊字符前加反斜杠 `\`（例如 `\*`、`\#`）。

## 其它技巧

- 内嵌 HTML：Markdown 中允许直接使用少量 HTML（如 `<kbd>`、`<abbr>`），但尽量避免复杂布局。
- 表情与自动链接（依赖渲染平台）。
- 使用带语言标记的代码块可以获得更好的语法高亮与行号支持（由渲染器决定）。
