# 一、创建一个 Vue 项目

> ## WARN
> 已安装 `^20.19.0 || >=22.12.0` 或更高版本的 `Node.js` \
> 可通过 *`nvm`* / *`fnm`* 命令管理多个`node`版本

确保你安装了最新版本的 `Node.js`，并且你的当前工作目录正是打算创建项目的目录。
``` bash
npm create vue@latest
```
