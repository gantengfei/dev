**方法一**

``` html
<a href="chrome://settings/clearBrowserData" target="_blank">清除缓存</a>
```

**方法二**
``` JavaScript
// 清除缓存通过刷新页面，强制浏览器重新加载资源文件，而不是从缓存中加载。
window.location.reload(true);

// 清除本地存储
// localStorage.clear();
// 删除特定的键值对
localStorage.removeItem('keyName');

// 清除会话存储
// sessionStorage.clear();
// 删除特定的键值对
sessionStorage.removeItem('keyName');
```
