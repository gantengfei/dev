
# 一、table合并行/列
**table表格的行`rowspan`、列`colspan`合并**

`colspan`属性：用于合并列。它定义了一个表格单元格应该横跨多少列。 \
`rowspan`属性：用于合并行。它定义了一个表格单元格应该横跨多少行。


# 二、HTML空格符号

**1.非断行空格 (`&nbsp;`)**

这是最常用的空格符号，表示一个非断行的空格。它可以防止文本在该空格处换行。 \
**示例：**`World&nbsp;World`**显示：**World&nbsp;World \
显示不会在World和World之间换行。

**2.半角空格 (`&ensp;`)**

表示一个半角空格，宽度约为一个中文字的一半。 \
**示例：**`Hello&ensp;World`**显示：**Hello&ensp;World \
会在Hello和World之间插入一个半角空格。

**3.全角空格 (`&emsp;`)**

表示一个全角空格，宽度约为一个中文字的宽度。 \
**示例：**`Hello&emsp;World`**显示：**Hello&emsp;World \
会在 Hello 和 World 之间插入一个全角空格。

**4.窄空格 (`&thinsp;`)**

表示一个窄空格，宽度比普通空格小。 \
**示例：**`Hello&thinsp;World`**显示：**Hello&thinsp;World \
会在Hello和B之间插入一个窄空格。

**5.零宽度空格 (`&zwnj;`)**

表示一个零宽度的空格，通常用于控制文本的排版而不占用空间。 \
在某些情况下，可以用来防止两个字符连在一起。

**6.零宽度连字 (`&zwj;`)**

用于在字符之间插入一个零宽度的连字符，通常用于某些语言的连字效果。
